# FlipQuiz
FlipQuiz is an online platform that helps you study information through interactive tools and games. Our mission is to help you practice and master what you're learning.

[LIVE DEMO](https://flipquiz.herokuapp.com/)
## Development setup

1. Install Node.js on your computer - [Node JS Installation - YouTube](https://www.youtube.com/watch?v=JINE4D0Syqw)
	
	Make sure both `node -v` and `npm -v` produce output with version number in console.
2. Install Yarn (Alternative to NPM package manager) - [How to install yarn](https://classic.yarnpkg.com/lang/en/docs/install)

	Make sure `yarn -v` produces output with version number in console.

3. Clone this repository and open it in VSCode. ([How to clone a repo](https://docs.github.com/en/desktop/contributing-and-collaborating-using-github-desktop/adding-and-cloning-repositories/cloning-and-forking-repositories-from-github-desktop#cloning-a-repository)) You will see suggested extensions pop up in the bottom right corner. Install them all.
4. Open a console in the root of the repository, or simply use the “Ctrl+`” keyboard shortcut in VSCode
5. Type `yarn install-all` in the console. You should see "Done" at the end of the process.
6. Type `yarn start-dev` in the console. A new tab in the browser will open, with app running at:

	http://localhost:3000/
7. Congrats! 😁 You can now start the development.
